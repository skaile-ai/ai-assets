# Base Chapter Template

All domain-specific templates extend this base structure. The base provides the common frame that every chapter must have.

```markdown
# {{chapter_title}}

> **Lernziele**: Nach dem Studium dieses Kapitels können Sie...
> 1. [Lernziel 1 — Bloom action verb: beschreiben, erklären, anwenden, analysieren]
> 2. [Lernziel 2]
> 3. [Lernziel 3]

## Fallvignette

> [Name], [Alter] Jahre, wird vorgestellt wegen [Beschwerde]...
> [2–3 Sätze klinische Beschreibung]
> [1–2 relevante anamnestische Angaben]

## [Inhaltliche Abschnitte — gemäß Vorlage]

<!-- Domain-specific sections are inserted here by the domain template -->

## Praxistipps

> **Praxistipp**: [Practical clinical pearl]
> **Praxistipp**: [Second pearl, if applicable]

## Kernaussagen

| # | Kernaussage |
|---|-------------|
| 1 | [Key takeaway 1] |
| 2 | [Key takeaway 2] |
| 3 | [Key takeaway 3] |

## Querverweise

- → siehe Kapitel [X]: [Bezug]
```

## Usage Notes

- **Lernziele**: Use Bloom's taxonomy action verbs (beschreiben, erklären, anwenden, analysieren, bewerten)
- **Fallvignette**: Must illustrate at least 2 teaching points from the chapter
- **Praxistipps**: Short, actionable, clinically relevant
- **Kernaussagen**: Maximum 5, each one sentence
- **Querverweise**: Link to related chapters by ID
