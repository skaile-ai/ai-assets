## Scaffold Guidance

When building the Knowledge Scaffold for a pharmacotherapy chapter:
- Always use INN (generic) drug names; include brand names only if clinically essential
- Include exact dosages with mg/kg/d format where applicable
- Include Fachinformation-verified UAW data with frequency categories
- Include Zulassungsstatus for the pediatric population (many drugs are off-label!)
- Include a monitoring protocol — this is critical for clinical practice
- Distinguish AWMF and NICE recommendations when they diverge
