# Market Sizing

## Three-Layer TAM

### Layer 1 — The Wedge (V1): Enterprise App Building

- Shadow IT remediation + internal tool development
- Target: mid-size companies (50–1,000 employees)
- ~500K mid-size companies in EU + US
- Each has 10–100 "shadow apps" (complex spreadsheets)
- Beyond shadow apps: untapped potential — processes that were never tooled because it wasn't economically viable
- **SAM: ~$5–10B**

### Layer 2 — Expansion (V2): Generalized Skill Pipelines + Open Sessions

- Same companies, same platform, new verticals via additional skill templates:
  - **Consulting:** question → research → hypothesis → validation → presentation
  - **Sales/Offers:** customer request → solution design → pricing → offer document
  - **Workshops:** request → agenda → research → slides → support materials
  - **Due diligence:** target profile → financial analysis → risk matrix → investment memo
- **Open sessions:** users get raw access to their IT-approved VM for freeform AI work — custom prompts, ad-hoc agents, exploratory analysis — without following a structured pipeline
- Enterprise AI automation market: ~$50B, growing ~35%/year

### Layer 3 — The Vision: Enterprise AI Process Platform

- Any structured intellectual process with artifacts, review cycles, and approvals encoded as skill pipelines
- Investment banking / M&A, Private Equity, tax advisory, HR, consulting, legal, insolvency administration (Insolvenzverwaltung), carve-outs
- Pipeline composition: output of one pipeline feeds into the next (research → presentation, due diligence → investment memo → board deck)
- Skill template marketplace: IT teams, partners, and third parties publish vertical-specific templates
- **TAM: $200B+** (enterprise knowledge work automation)

## Key Insight

Each layer uses the same core infrastructure (VM per session, skill templates, artifact pipeline, human oversight). Not three products — three markets unlocked with one platform.

The transition from V1 to V2 is not a product pivot — it is the same platform with additional skill templates. The software vertical proves the model; subsequent verticals are configuration, not construction.

## Target Segments (V1)

1. **Mid-size companies** (50–1,000 employees): custom internal processes, shadow IT, AI pressure from leadership
2. **Consulting companies**: need custom tools for client projects (e.g., McKinsey Wave, PostXL SteerEx)
3. **Internal (data2impact)**: dogfooding — build and manage apps on the framework
