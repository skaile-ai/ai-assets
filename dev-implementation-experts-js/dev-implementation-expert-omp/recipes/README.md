# Recipes

Store coding recipes for recurring omp tasks here. Each recipe should include frontmatter (name, description, libraries_used) and markdown instructions with code examples.

## Categories
- **rpc-** — RPC sidecar integration recipes
- **skill-** — Skill authoring recipes
- **embed-** — Embedding patterns for host apps
- **config-** — Configuration recipes

## Recipes

| Recipe | Description |
|--------|-------------|
| `rpc-node-sidecar.md` | Spawn and manage omp as an RPC child process in Node.js |
| `rpc-sse-bridge.md` | Forward omp events to browser via SSE (simple/legacy pattern) |
| `embed-typed-action-protocol.md` | **Recommended** — Typed Action Protocol: bidirectional SkillActions + ConsumerResponses, state machine, input forms, approval flows |
| `skill-authoring.md` | Author SKILL.md files for the omp skill system |
| `config-project-setup.md` | Configure omp for a new project |
