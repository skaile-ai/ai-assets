# concept-2-experience-3-screens CLI

## Trigger

Invoke with: "Specify screens" or "Plan the screens for my app"

## Prerequisites

- `_concept/2_experience/2_features/**/*.md` must exist (run `concept-2-experience-2-features` first)
- `_concept/1_discovery/3_brand/tokens.json` recommended (run `concept-1-discovery-3-brand`)
- `_concept/3_blueprint/1_techstack/stack.md` recommended (run `concept-3-blueprint-1-techstack`)
- `_concept/3_blueprint/3_datamodel/postxl-schema.json` recommended (run `concept-3-blueprint-3-datamodel`)

## Output

- `_concept/2_experience/3_screens/00_layout/shell.md` — app layout
- `_concept/2_experience/3_screens/<NN_group>/<screen>.md` — one per screen
- Updates `screens:` in feature frontmatter (feedback loop)
