# Layout: App Shell

## Structure
- Top navbar: logo (left), user avatar + dropdown (right)
- Sidebar: navigation links (Dashboard, Tasks, Settings)
- Main content area: routed views
- No footer

## Responsive
- Desktop (1024+): sidebar visible, content fills remaining width
- Tablet (768-1023): sidebar collapses to icons
- Mobile (<768): sidebar hidden, hamburger menu in navbar

- Background: #ffffff (colors.background)
- Sidebar surface: #f8fafc (colors.surface)
- Primary accent: #6366f1 (colors.primary)
- Font: Inter (fonts.body)
