# Layout: App Shell

## How the App is Organized
- A top bar with the app logo on the left and the user's profile on the right
- A sidebar with navigation links to the main areas: Dashboard, Tasks, Settings
- The main content area where each screen appears

## On Different Devices
- On desktop: sidebar is always visible, content fills the rest of the screen
- On tablets: sidebar shows only icons to save space
- On phones: sidebar is hidden behind a menu button in the top bar
