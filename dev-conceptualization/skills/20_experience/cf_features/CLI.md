# cf_concept_functionality_features CLI

## Trigger

Invoke with: "Plan the features for my app" or "Define features"

## Prerequisites

- `_concept/01_project/brief.md` must exist (run `cf_concept_overview` first)

## Output

- `_concept/03_features/<NN_group>/<feature>.md` — one file per feature, grouped in numbered folders

## Next Step

After human approval → optionally run `cf_concept_functionality_behaviors` for behavioral specs, then define brand (`cf_concept_brand_visual`), tech stack (`cf_concept_techstack`), then run `cf_concept_datamodel`
