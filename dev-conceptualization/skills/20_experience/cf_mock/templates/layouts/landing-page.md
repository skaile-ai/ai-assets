# Landing Page Template

When the user requests a Landing Page or Marketing Site, strictly adhere to this bold, visual layout structure.

## Core Structure
- **Floating Header (Navigation):** Fixed at the top, typically transparent or blurred (`backdrop-blur-md`), containing brand logo, simple navigation links, and a primary CTA (Call to Action) button.
- **Hero Section:** The most critical area. Needs a bold `H1` headline (e.g., `text-5xl md:text-7xl font-extrabold`), a descriptive `p` subheadline, and two primary actions (e.g., "Get Started" [Primary Button], "View Demo" [Secondary Button]).
- **Features Grid:** A section highlighting 3-4 key features using cards with large, expressive icons (Iconify) and punchy text.
- **Social Proof / Testimonials:** A visually distinct section (e.g., darker background) containing user quotes or logos of "trusted by" companies.
- **Footer:** Simple, organized links and copyright information.

## Aesthetic Requirements
- **Gradients and Glows:** Use subtle background radial gradients or blurred colorful orbs behind the Hero section to create a premium, modern feel.
- **Generous Spacing:** Use excessive whitespace. Sections should have massive padding (e.g., `py-24` or `py-32`).
- **Typography:** Headlines must tightly track (`tracking-tight`), and buttons must feel substantial (`px-8 py-4`).

## Anti-Patterns
- **No Dense Data:** Do not use tables or complex forms on a landing page.
- **No Dead Ends:** Every section should ideally drive the user toward the primary CTA.
