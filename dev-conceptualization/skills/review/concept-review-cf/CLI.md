# cf_quality_review CLI

## Trigger

Invoke with: `/concept-review` or "Review the concept status" or "What's missing from my concept docs?"

## Output

A status table printed in the conversation — no files written.

## When to Use

- After running `cf_orchestrator` to verify all phases completed
- Before running `cf_concept_mock` to confirm screen specs exist
- Before running `cf_test_e2e` to confirm mockups exist
- Any time you want a birds-eye view of progress
